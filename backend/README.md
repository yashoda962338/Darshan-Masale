# Darshan Masale Backend API

Enterprise backend for Darshan Masale - Premium Indian Spice Brand

## 🚀 Tech Stack

- **Runtime**: Node.js
- **Framework**: Express.js
- **Database**: MySQL
- **ORM**: mysql2 (native)
- **Authentication**: JWT
- **File Upload**: Multer + Cloudinary
- **Security**: Helmet, CORS, Rate Limiting
- **Logging**: Morgan + Custom Logger
